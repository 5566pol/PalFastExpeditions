# 幻兽帕鲁全自动远征

由强大的Mimo-v2.5-pro代写（）
自动快速完成帕鲁远征，使用Tesseract OCR 图像识别屏幕文字并点击操作。

## 环境要求

- Windows 10/11
- Tesseract OCR

## 安装 Tesseract OCR

1. 下载 Tesseract OCR 安装包：
https://github.com/UB-Mannheim/tesseract/wiki
2. 安装时勾选 **Chinese Simplified** 语言包
3. 在程序界面中修改 Tesseract 路径

## 使用方法

1. 打开程序后，在界面顶部选择远征目的地
2. 在界面底部选择 Tesseract 路径
3. 幻兽帕鲁里走到远征机面前，确保站在那按下F能显示远征界面，然后什么都不要动
4. 确保游戏为全屏窗口或窗口模式，窗口保证在前台
5. 按 **PageDown（默认）** 键开始/停止自动化流程
